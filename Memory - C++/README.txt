Game of Memory made using c++

2D Array populated with 2 of each number from 1-8 randomly

User inputs a guess of a number in the form of #row,#col (for example 1,3)
Game displays the number of their guess.
User guesses again, displaying the number of their guess, as well as if they were correct or wrong, and their percentage of correct guesses.

Program runs until player guesses all matching pairs.

User can input 9,9 to see the board (included for debugging purposes)
